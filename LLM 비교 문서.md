# 모델 비교·선정 보고서

## 1. 비교 목적

회의록 요약 업무에 가장 적합한 LLM을 선정하기 위해 GPT-5, Claude 3.5 Opus, Gemini 3.1 Pro를 비교하였다.

## 2. 비교 대상 모델 및 사용 환경

| 모델명     | 버전       | 사용 환경  |
| ------- | -------- | ------ |
| ChatGPT | GPT-5    | 학습 네이토 |
| Claude  | 3.5 Opus | 학습 네이토 |
| Gemini  | 3.1 Pro  | 학습 네이토 |

## 3. 테스트 시나리오

**업무 과업**
회의록 요약

**원문**

* 5월 25일 프로젝트 회의
* 코디세이 앱 출시일을 6월 5일로 확정
* 로그인 오류 수정
* 디자인 시안 제출
* 테스트는 5월 30일까지 완료
* 서버 비용 점검

**요구 출력**

* 결정사항
* Action Items
* 리스크

## 4. 평가 기준

1. 정확성
2. 한국어 자연스러움
3. 형식 준수
4. 긴 문맥 유지
5. 업무 활용성

## 5. 평가 결과

| 평가 항목  | GPT-5 | Claude 3.5 Opus | Gemini 3.1 Pro |
| ------ | ----- | --------------- | -------------- |
| 정확성    | 5     | 5               | 4              |
| 자연스러움  | 5     | 5               | 4              |

## 6. 최종 선정 모델

**ChatGPT (GPT-5)**

## 7. 선정 이유

* 회의록 핵심 정보를 누락 없이 정리함
* 구조화 능력이 뛰어나 업무 효율 증가
* 형식 준수 능력이 우수함
* 문맥 유지가 안정적임
